Описание бота:
- Бот который парсинт сайт РР4 для Норвежского моря на приманки
Что должен уметь бот:
- по запросу парсить сайт рр4 и записывать в файл json
- по запросу выводить информацию о приманках для мелкой рыбы
- по запросу выводить информацию о приманках для трофейной рыбы
- по запросу выводить информацию о приманках для другой рыбы
- выводить информацию в определенное время для всех рыб
Описание работы:
задействовать кнопки /start /help для активации бота
